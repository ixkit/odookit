All Odoo modules can be load in online code editor, write code, hot-reload frontend UIs, easily and quickly!

Usage

    Start Code Server from the console page from this module, once start success, browser auto navigate to :http://localhost:3030/ ; The code editor was provide by the open source Code Server

Additional Step: Manual install the Code Server

  • Download the Code server program from the repository or clone the source code and build the Code Sever package.

  • Put the Code Sever package into the moudule directory './kit_code/vendors/', it should be './kit_code/venders/code_server'
